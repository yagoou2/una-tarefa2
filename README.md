# 📘 Lista de Exercícios em Java

## 🧠 Resumo das Atividades

Nesta tarefa, foram desenvolvidos programas em Java com o objetivo de praticar conceitos fundamentais da linguagem, incluindo entrada de dados, estruturas de controle, manipulação de strings e uso de funções matemáticas.

---

## 🖥️ 1. Impressão de ASCII Art

Foi criado um programa para exibir uma figura no console utilizando caracteres especiais.

### Conceitos aplicados:

* Uso de `System.out.println`
* Escape de caracteres (`\"`)
* Organização da saída no terminal

---

## 🌍 2. Cálculo de Distância entre Coordenadas

Desenvolvimento de um programa que calcula a distância entre dois pontos na superfície da Terra utilizando uma fórmula trigonométrica.

### Conceitos aplicados:

* Funções matemáticas (`Math.sin`, `Math.cos`, `Math.acos`)
* Conversão de graus para radianos (`Math.toRadians`)
* Uso de `double` para maior precisão
* Entrada de dados com `Scanner`

---

## 🔤 3. Contador de Caracteres

Criação de um programa que analisa uma string fornecida pelo usuário e contabiliza:

* Letras
* Números
* Espaços
* Outros caracteres

### Conceitos aplicados:

* Estrutura de repetição `for`
* Manipulação de strings (`length`, `charAt`)
* Classe `Character` (`isLetter`, `isDigit`, `isWhitespace`)
* Estruturas condicionais (`if / else if`)

---

## 🎯 4. Sistema de Múltipla Escolha

Implementação de um programa interativo com uma pergunta de múltipla escolha, contendo 5 alternativas (A–E), com limite de até 3 tentativas.

### Funcionalidades:

* Exibição da tentativa atual
* Validação da resposta utilizando `switch`
* Repetição automática em caso de erro
* Tratamento de opção inválida (não contabiliza tentativa)
* Mensagem indicando em qual tentativa o usuário acertou
* Mensagem final caso o usuário erre todas as tentativas

### Conceitos aplicados:

* Estrutura `switch-case`
* Loop `do-while`
* Controle de fluxo com variáveis booleanas
* Contador de tentativas
